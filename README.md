# 카메라 기반 인공지능 모델 실습

이 프로젝트는 카메라를 이용한 이미지 캡처와 인공지능 모델 실습을 위한 기본 환경을 제공합니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 사용 방법

1. 카메라 캡처 테스트:
```bash
python camera_capture.py
```

## 주요 기능

- 카메라를 통한 이미지 캡처
- 캡처된 이미지 표시
- 이미지 전처리 기능

## 주의사항

- 카메라가 연결되어 있어야 합니다.
- Python 3.7 이상 버전이 필요합니다.
